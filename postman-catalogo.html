Peticiones Postman - Microservicio de Catálogo de Libros

GET Obtener todos los libros

http://localhost:8081/api/books

Obtiene la lista completa de libros disponibles en el catálogo.

Respuesta de ejemplo:

[ { "id": 1, "title": "El Quijote", "author": "Miguel de Cervantes", "isbn": "978-8497593798", "price": 19.99, "stock": 50 }, { "id": 2, "title": "Cien años de soledad", "author": "Gabriel García Márquez", "isbn": "978-8497592208", "price": 15.99, "stock": 30 } ]

GET Obtener libro por ID

http://localhost:8081/api/books/{id}

Obtiene la información detallada de un libro específico por su ID.

Respuesta de ejemplo:

{ "id": 1, "title": "El Quijote", "author": "Miguel de Cervantes", "isbn": "978-8497593798", "price": 19.99, "stock": 50, "description": "La obra maestra de la literatura española", "category": "Novela", "publicationDate": "1605-01-01" }

POST Crear nuevo libro

http://localhost:8081/api/books

Crea un nuevo libro en el catálogo.

Cuerpo de la petición:

{ "title": "Nuevo Libro", "author": "Autor Ejemplo", "isbn": "978-1234567890", "price": 24.99, "stock": 100, "description": "Descripción del nuevo libro", "category": "Ficción", "publicationDate": "2024-01-01" }

Respuesta de ejemplo:

{ "id": 3, "title": "Nuevo Libro", "author": "Autor Ejemplo", "isbn": "978-1234567890", "price": 24.99, "stock": 100, "description": "Descripción del nuevo libro", "category": "Ficción", "publicationDate": "2024-01-01" }

PUT Actualizar libro

http://localhost:8081/api/books/{id}

Actualiza la información de un libro existente.

Cuerpo de la petición:

{ "title": "Libro Actualizado", "author": "Autor Actualizado", "price": 29.99, "stock": 75, "description": "Nueva descripción del libro" }

Respuesta de ejemplo:

{ "id": 1, "title": "Libro Actualizado", "author": "Autor Actualizado", "isbn": "978-8497593798", "price": 29.99, "stock": 75, "description": "Nueva descripción del libro", "category": "Novela", "publicationDate": "1605-01-01" }

DELETE Eliminar libro

http://localhost:8081/api/books/{id}

Elimina un libro del catálogo por su ID.

Respuesta de ejemplo:

{ "message": "Libro eliminado exitosamente", "id": 1 }

GET Buscar libros por título

http://localhost:8081/api/books/search?title={título}

Busca libros por título (búsqueda parcial).

Respuesta de ejemplo:

[ { "id": 1, "title": "El Quijote", "author": "Miguel de Cervantes", "isbn": "978-8497593798", "price": 19.99, "stock": 50 } ]

GET Obtener libros por categoría

http://localhost:8081/api/books/category/{categoría}

Obtiene todos los libros de una categoría específica.

Respuesta de ejemplo:

[ { "id": 1, "title": "El Quijote", "author": "Miguel de Cervantes", "isbn": "978-8497593798", "price": 19.99, "stock": 50, "category": "Novela" } ]
